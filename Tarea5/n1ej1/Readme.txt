Para compilar por comandos lo primero que hay que hacer es revisar las variables de entorno. 

LLuego abrimos cmd y se escribe:
 	cd "...src\Tarea5\.."//ruta


Por ultimo se compilan los archivos en el siguiente orden:
	javac src/Tarea5/n1ej1/ListarDirectorio.java
	javac src/Tarea5/n1ej1/Main.java
